# [middle.messenger.praktikum.yandex]

Чат для общения. Приятный интерфейс, удобный функционал, ничего лишнего.

## Команды проекта 
Запуск разработки через браузер 
 - `npm run dev` 
-Сборка приложения
 - `npm run build` 
-Запуск проекта на localhost:
 - `npm run start`
